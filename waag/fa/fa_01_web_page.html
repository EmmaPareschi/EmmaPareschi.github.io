Chrome - Developer Tools
These tools will allow you to not only see the HTML, but also the CSS that applies to view elements in that HTML document.
To use Chrome's developer tools:
- Open Google Chrome
- Navigate to the web page you would like to examine
- Click the icon with three lines in the upper right corner of the browser window
- From the menu, hover over "More tools" and then click "Developer tools". This will open a window that show the HTML source code on the left of the pane and the related CSS on the right.
- Alternatively, if you right click an element in a web page and select "Inspect" from the menu that appears, Chrome's developer tools will pop up and the exact element you have selected will be highlighted in the HTML with the corresponding CSS shown to the right.
