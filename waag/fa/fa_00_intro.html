INTRO

To keep my notes available whenever...or at least where there is a network, I created this page and decided to try github web server. I followed this intructions: pages.github.
From the option I downloaded github for Mac.

How I use it.

I saved my github master copy in Documents and I use Sublime Text 2 to edit the html page.
Every time I change a file, from github Desktop app I sync, commit and sync again.
My github page server: https://github.com/EmmaPareschi/EmmaPareschi.github.io .

How to update my pages
1- git pull
2- git status
3- git add .a
4- git commit -am "description of the change you update"
5- git push

Problem 1
After few months I was not updating the site I tried to push and I got the following error text: "remote: Permission to EmmaPareschi/EmmaPareschi.github.io.git denied to emmapa. fatal: unable to access 'https://github.com/EmmaPareschi/EmmaPareschi.github.io.git/': The requested URL returned error: 403".
I fixed the error following the steps:
1- edit .git/config file under your repo directory
2- change the url from "https://github.com/EmmaPareschi/EmmaPareschi.github.io.git" to "https://EmmaPareschi@github.com/EmmaPareschi/EmmaPareschi.github.io.git".
At the first push, after the changes, github asked the password of my github account.
It worked. Happy me.
